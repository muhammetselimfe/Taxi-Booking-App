# Taxi-Booking-App

## Installation
1. Clone this repository to your local machine.
2. Install backend dependencies and start application by running:
````
npm install
````
````
npm start
````

## Technologies Used
- Node.js (runtime environment)
- Express (web framework for Node.js)
- Jest (E2E and Unti testing)
